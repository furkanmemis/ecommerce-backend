# ecommerce-backend


##  Project Purpose

This project is a **personal backend development playground** created to improve my skills in **Java and Spring Boot**.

The main goal is to gain hands-on experience with:
- Building RESTful APIs
- Designing clean and modular backend architectures
- Implementing JWT-based authentication and middleware
- Understanding request lifecycle, filters, and annotations
- Working with real-world concepts such as users, roles, stock, and notifications

Instead of focusing on a complete production-ready e-commerce system, this project focuses on **learning by building**, experimenting with different approaches, and understanding **how things work internally** in a Java backend application.

The project evolves over time as new concepts are learned and applied.
